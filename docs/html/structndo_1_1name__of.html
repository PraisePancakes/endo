Detailed Description

template<typename T>
struct ndo::name_of< T >

Compile-time type name mapping.

Provides a constexpr string literal representing the name of a type T. By default, the value is an empty string.

Specializations are provided for common fundamental types.

Usage example:

constexpr auto int_name = ndo::name_of<int>::value; // "int"
