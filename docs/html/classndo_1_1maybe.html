Detailed Description

template<typename T>
struct ndo::maybe< T >

A container that may hold a value or nothing.

Forward declaration of maybe template.

The maybe<T> type is used to represent an optional value of type T. It can either hold a valid value (T) or represent an empty state using ndo_null_t. It provides utilities for value access, transformation, and monadic chaining.

Template Parameters
TThe type of the value that may be held.
