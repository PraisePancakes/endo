Detailed Description

template<typename Ret, typename... Args>
struct ndo::function_traits_meta< Ret, Args >

Extracts type traits from callable types.

Provides information such as return type, argument types, and arity (number of arguments) for various callable types, including function pointers, member function pointers, functors (lambdas), and std::function.

Usage example:

auto lambda = [](int a, double b) -> bool { return a < b; };
using traits = ndo::function_traits<decltype(lambda)>;
static_assert(std::is_same_v<traits::return_t, bool>);
static_assert(traits::arity == 2);
using first_arg = traits::arg_t<0>; // int
