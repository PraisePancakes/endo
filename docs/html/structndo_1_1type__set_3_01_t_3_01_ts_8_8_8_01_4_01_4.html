◆ filter

using ndo::type_set< Ts >::filter

Filter the type_set based on a compile-time predicate Cond.

Only types T for which Cond::template operator()<T>() returns true are included in the resulting type_set.

Template Parameters
CondA callable with a templated operator() that takes a type and returns bool.
