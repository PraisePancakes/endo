Detailed Description

Y Combinator utility to enable anonymous recursion for lambdas.

Transforms a curried callable into a callable accepting all arguments at once.

Curries a callable so it can be invoked with partial arguments in multiple calls.

The Y combinator allows defining recursive lambdas without naming the lambda.

auto factorial = ndo::make_y_combinator([](auto self, int n) -> int {
return n <= 1 ? 1 : n * self(n - 1);
});
int result = factorial(5); // 120

A callable that supports recursion via the first argument.

Transforms a callable f taking multiple arguments (int, char, bool) into a curried callable like g(int)(char)(bool).

Example usage:

auto f = [](int x, char y, bool z) { return x + y + (z ? 1 : 0); };
auto curried_f = ndo::curry(f);
int result = curried_f(1)('a')(true);

Converts a curried function f like f(int)(int) into a function g(int, int).

Example usage:

};
auto uncurried_add = ndo::uncurry(curried_add);
int result = uncurried_add(1, 2); // 3
